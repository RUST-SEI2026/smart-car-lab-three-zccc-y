# 需求3：支持不同的车辆类型
**1. 跑车**

速度很快，指令控制行为遵循下述描述：
+ M：前进，1次移动2格(先前进1格再前进1格)
+ L：先左转90度，再前进1格
+ R：先右转90度，再前进1格
+ 掉头操作与普通车辆相同
+ 与普通车辆一样，支持B和F两个状态的叠加
 

|  | M | L | R |
| ---- | ----- | ---- | ---- |
| normal | 前进2格（1次移动1格） | 左转90度，再前进1格 | 右转90度，再前进1格 |
| B   | 后退2格（1次移动1格） | 右转90度，再后退1格 | 左转90度，再后退1格 |
| F   | 前进4格（1次移动1格） | 先前进1格再左转90度，而后再前进1格 | 先前进1格再右转90度，然后再前进1格 |
| B&F | 后退4格（1次移动1格） | 先后退1格再右转90度，而后再后退1格 | 先后退1格再左转90度，然后再后退1格 |


**2. Bus**

Bus车体较长，长度占2格，它的坐标是车头的坐标。为了行车安全，指令控制行为遵循下述描述：
+ M：前进，1次移动1格
+ L：先前进1格，再左转90度
+ R：先前进1格，再右转90度
+ 掉头操作与普通车辆相同
+ 与普通车辆一样，支持B和F两个状态的叠加

|  | M | L | R |
| ---- | ---- | ---- | ---- |
|     | 前进1格          | 前进1格，再左转90度 | 前进1格，再右转90度 |
| B   | 后退1格          | 后退1格，再右转90度 | 后退1格，再左转90度 |
| F   | 前进1格，再前进1格 | 前进1格，再前进1格，而后左转90度 | 前进1格，再前进1格，而后右转90度 |
| B&F | 后退1格，再后退1格 | 后退1格，再后退1格，而后右转90度 | 后退1格，再后退1格，而后左转90度 |
